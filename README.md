- 💬 **AI Chat** — Ask anything and get intelligent replies powered by LLaMA 3.3 70B

- ⚡ **Instant Replies** — Fast static responses for common greetings
- 🛡️ **Rate Limiting** — Prevents spam with built-in cooldowns
- 🏓 **Ping Command** — `/ping` slash command to check if the bot is alive

## 🆕 New to Discord? Start Here!

Never used Discord before? No worries — follow these simple steps:

### Step 1 — Create a Discord Account
1. Go to **[https://discord.com](https://discord.com)**
2. Click **"Open Discord in your browser"** or download the app
3. Click **"Register"** and fill in your:
   - Email address
   - Username (your display name)
   - Password
   - Date of birth
4. Verify your email by clicking the link Discord sends you

---

### Step 2 — Create Your Own Server
1. After logging in, look at the **left sidebar**
2. Click the **➕ (plus icon)** at the bottom of the server list
3. Choose **"Create My Own"**
4. Select **"For me and my friends"**
5. Give your server a name and click **"Create"**

> ✅ You now have your own Discord server!

---

### Step 3 — Add adiBot to Your Server
1. Click this invite link:
   👉 **[Add adiBot](https://discord.com/oauth2/authorize?client_id=1478646599084408853&permissions=274877991936&scope=bot%20applications.commands)**
2. A Discord page will open — **log in if prompted**
3. Under **"Add to Server"**, select the server you just created
4. Click **"Continue"**
5. Review the permissions and click **"Authorize"**
6. Complete the CAPTCHA if asked

> 🎉 adiBot is now in your server!

---

### Step 4 — Start Chatting with adiBot
1. Open your server and click on any **text channel** (e.g., `#general`)
2. Type a message and hit **Enter** — adiBot will reply!

#### 💡 Try these commands:
| Message | Response |
|---|---|
| `hello` | Quick greeting reply |
| `name?` | Bot tells you its name |
| `delete chat` | Clears the channel (admin only) |
| `/ping` | Bot responds with 🏓 Pong! |
| Anything else | AI-powered reply from LLaMA 3.3 |

## 🔒 Permissions Explained

When adding the bot, it will request these permissions:

| Permission | Why it needs it |
|---|---|
| **Read Messages** | To see what you type |
| **Send Messages** | To reply to you |
| **Manage Messages** | To delete chat when asked |
| **Use Slash Commands** | For `/ping` and future commands |
